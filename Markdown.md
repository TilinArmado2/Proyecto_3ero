# Introducción a Markdown

## ¿Qué es Markdown?

Markdown es un lenguaje de marcado ligero que permite dar formato al texto usando una sintaxis simple basada en caracteres como `#`, `*`, `-` y `[]`.

Se utiliza mucho para escribir documentación porque permite crear textos estructurados sin tener que usar HTML directamente.

Markdown se usa en:

- GitHub
- documentación de proyectos
- wikis
- blogs técnicos
- notas de desarrollo

## Cómo funciona Markdown

Markdown permite escribir texto en formato plano y luego convertirlo a HTML u otros formatos visuales mediante un procesador.

Por ejemplo si escribimos:

## Hola mundo
**Este texto está en negrita**

el procesador lo interpreta y lo muestra como un título y texto en negrita.

## Títulos

Los títulos se crean usando el símbolo `#`. Cuantos más `#` se usen, menor será el nivel del título.

# Título nivel 1
## Título nivel 2
### Título nivel 3
#### Título nivel 4

## Negrita y cursiva

Para dar formato al texto se utilizan asteriscos.

**Texto en negrita**
*Texto en cursiva*

Markdown permite crear listas fácilmente.

Lista con viñetas:

- Elemento 1
- Elemento 2
- Elemento 3

Lista numerada:

1. Primer elemento
2. Segundo elemento
3. Tercer elemento
## Enlaces

Los enlaces se escriben de la siguiente forma:

[Texto del enlace](https://ejemplo.com)

Ejemplo:

[Ir a GitHub](https://github.com)

## Imágenes

Las imágenes se insertan de forma parecida a los enlaces.

![Texto alternativo](imagen.png)

## Código

Markdown también permite mostrar código.

Código en línea:

```
Usar `print()` en Python
```

Bloque de código:

```python
print("Hola mundo")
```

## Conclusión

Markdown es una herramienta muy útil para escribir documentación de forma clara y rápida, su lenguaje es relativamente sencillo aunque por lo que estuve viendo puede llegar a ser confuso, al menos en la parte de los links para fotos y imagenes.
